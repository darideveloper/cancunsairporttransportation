## Requirements
### Requirement: Asset Usage
The `BannerStats` component MUST use the Astro `<Image />` component for icons instead of React Icons.
#### Scenario: Rendering
- GIVEN the component is rendered
- THEN it must render 3 `<img />` tags generated by Astro
- AND it must NOT render any `<svg>` tags from `react-icons`

### Requirement: Accessibility Metadata
Each stat image MUST have localized `alt` and `title` attributes.
#### Scenario: Attributes present
- GIVEN a stat item "clients"
- THEN the image MUST have `alt` text fetched from `bannerStats.stats.clients.imageAlt`
- AND the image MUST have `title` text fetched from `bannerStats.stats.clients.imageTitle`
