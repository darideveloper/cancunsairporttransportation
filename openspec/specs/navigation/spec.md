## Requirements
### Requirement: Blog Link in Header
The header navigation MUST include a link to the blog.

#### Scenario: Viewing the header
- **Given** the user is on any page
- **When** looking at the main navigation menu
- **Then** a "Blog" link should appear after the "Cancun to Tulum Shuttle" link
- **And** it should link to the blog index page (localized)

### Requirement: Blog Link in Footer
The footer services section MUST include a link to the blog.

#### Scenario: Viewing the footer SERVICES column
- **Given** the user is on any page
- **When** looking at the "Services" column in the footer
- **Then** a "Blog" link should appear after the "Cancun Airport Taxi" link
- **And** it should link to the blog index page (localized)
