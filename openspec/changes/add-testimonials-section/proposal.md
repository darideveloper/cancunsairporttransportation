# Proposal: Add Testimonials Section

## Goal
Add a new testimonials section to the project to build trust with users. This section will feature client feedback with photos, names, ratings, and testimonial text.

## Scope
- New `Testimonial` molecule component.
- New `Testimonials` organism component.
- Integration into the translation system.
- Addition of client photos to assets.
- Integration of the section into the homepage.

## Out of Scope
- Visual styling (HTML structure and components only for now).
- Dynamic data (data will be fixed and integrated into the translation files).
