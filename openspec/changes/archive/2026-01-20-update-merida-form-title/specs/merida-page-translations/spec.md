# merida-page-translations Specification Delta

## ADDED Requirements

### Requirement: Merida Page Title
The Merida transportation page MUST show the correct localized title.

#### Scenario: Merida English Title
Given the user is on the Merida transportation page in English
Then the page title (rendering in the BookingForm) MUST be "Transportation from Cancun to Merida".

#### Scenario: Merida Spanish Title
Given the user is on the Merida transportation page in Spanish
Then the page title (rendering in the BookingForm) MUST be "Transporte de Cancún a Mérida".
