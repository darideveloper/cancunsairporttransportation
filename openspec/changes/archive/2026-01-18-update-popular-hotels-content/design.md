# Design

## Content Strategy
The "Popular Hotels" section is designed to boost SEO and user relevance by listing specific "Transportation from Cancun Airport to [Hotel]" phrases. This matches common user search intent.

## Data Structure
The `popularHotels` object in translation files will follow this structure:
```json
"popularHotels": {
    "title": "...",
    "viewMore": "...",
    "viewMoreLink": "/destinations", // or /es/destinos
    "items": [
        "Transportation from Cancun Airport to ...",
        ...
    ]
}
```
