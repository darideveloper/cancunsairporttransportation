# Translation Updates

## ADDED Requirements

### Requirement: Summary Translations
- The application MUST include translation keys for the vehicle summary card.
#### Scenario: English Locale
- Given the locale is English
- The summary keys should return English text (e.g., "Origin: ...").
#### Scenario: Spanish Locale
- Given the locale is Spanish
- The summary keys should return Spanish text (e.g., "Origen: ...").
