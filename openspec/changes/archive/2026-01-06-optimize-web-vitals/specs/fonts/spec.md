# Spec: Font Optimization

## ADDED Requirements

### Requirement: Single Source of Truth for Fonts
The system SHALL use a single method for loading fonts to avoid double-downloads and race conditions.

#### Scenario: Remove Redundant Preloads
- **Given** the application uses `@fontsource` for Inter and Metropolis
- **When** `Layout.astro` is rendered
- **Then** it SHALL NOT contain manual `<link rel="preload">` tags pointing to the `public/fonts/` directory
- **And** it SHALL rely on the CSS imports to trigger font loading.

### Requirement: Font Display Strategy
Fonts SHALL use `font-display: swap` (or fallback) to ensure text is visible immediately.

#### Scenario: Verify Font Display
- **Given** the font face definitions generated by `@fontsource`
- **When** inspected in the browser
- **Then** `font-display` should be set to `swap` (default for most Fontsource packages) or explicitly configured.
