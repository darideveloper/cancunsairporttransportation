# Add Blog Links to Navigation

## Modification Summary
Add a link to the Blog in both the main header navigation and the footer services section to improve content discoverability.

## Problem Statement
Users may have difficulty finding the blog section, which contains valuable information, as it is not currently linked in the main navigation or footer.

## Proposed Solution
- Add "Blog" link to the header after "Cancun to Tulum Shuttle".
- Add "Blog" link to the footer "Services" column after "Cancun Airport Taxi".
